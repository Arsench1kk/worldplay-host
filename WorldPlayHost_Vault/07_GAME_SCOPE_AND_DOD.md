Build vertical playable slices. Do not create empty shells for every game first.

## Judge-Friendly Games

These games are important because judges already understand them or can learn them instantly:

```txt
The Hat / Shapka
Truth or Dare
Spy / Impostor
Taboo / Don't Say It
Charades / Explain-the-word
Never Have I Ever lite
Would You Rather lite
```

They should support:

```txt
one-device local party mode
remote room code mode
AI player / AI prompt generator mode
```

## Final Realistic Target

```txt
20 real games in catalog
4-7 playable games
Spin the Globe
AI Host
Voice button if easy
Room mode only if stable
```

Better:

```txt
4 polished playable games > 8 broken prototypes
2 familiar judge-friendly games > 2 obscure cultural games if evaluation clarity is the risk
```

## Game DOD

Each playable game needs:

```txt
Start state
Short rules
Playable loop
Visible AI Host role
Score/result
Replay
Local fallback content
Mobile-safe layout
```
