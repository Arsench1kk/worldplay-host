# Risks And Fallbacks

## AI API Down

Actions:

```txt
Show AI Simulation Mode
Use data/cultures.ts fallbackHostIntro
Use culture facts[0]
Use local game recommendations
Use local game content
Return HTTP 200 from /api/ai
```

Do not spend more than 15 minutes trying to revive provider access.

## Theme Or Scope Starts Drifting

Actions:

```txt
Return to WorldPlay Host
Return to Spin the Globe
Return to same-device pass-and-play
Cut multiplayer and catalog expansion
```

## Visual Looks Weak

Actions:

```txt
Make background darker
Use passport stamp
Use AI Host panel
Use 3 strong badges
Use larger result cards
Add simple spin/glow animation
```

## Game Logic Not Ready

Actions:

```txt
Ship lite mode
Add "Inspired by" label if rules are simplified
Make one complete round work
Add replay
Move on
```

## Key Feature Broken 30 Minutes Before Deadline

Actions:

```txt
Hide broken feature
Use deterministic fallback
Record working happy path
Mention prototype mode in README
```

## Internet Unstable

Actions:

```txt
Run localhost
Record local browser
Upload video to Google Drive or YouTube unlisted
Submit repo + video + note
```
