# Почему это может выиграть

## 1. Great use of AI

AI не просто генерит текст. Он выполняет 4 роли:

```txt
AI Host        -> объясняет правила как живой ведущий
AI Curator     -> рекомендует игру под количество людей
AI Generator   -> создает слова, вопросы, задания, фразы
AI Player      -> может играть вместо отсутствующего человека
```

Это глубже, чем “AI Dungeon Master”.

## 2. Original game

Большинство сделает:

```txt
AI RPG
AI карточки
AI NPC
AI quiz
```

А ты делаешь:

```txt
социальную игру для реальной компании
путешествие по играм разных культур
AI-ведущего вместо скучных инструкций
```

Это запоминается.

## 3. UI/UX

Визуальный стиль:

```txt
Globe / Passport / Party table / Cultural cards
```

Не делай скучный каталог. Делай как “паспорт игр мира”.

## 4. Gameplay

30-секундный тест будет сильный:

```txt
выбрал 5 игроков
AI предложил игру
AI объяснил правила
нажал Start
пошел таймер
команда угадывает слова
score растет
```

Это сразу играется, а не читается.

# Какие 3 игры реализовать

## 1. The Hat / Шапка

Это главная игра. Она самая “gameplay”.

### Core loop

```txt
Создать 2 команды
AI генерирует 30 слов
Раунд 1: объясняй словами
Таймер 45 секунд
Команда угадывает
Score +1
Следующая команда
```

Для MVP можно сделать только один режим:

```txt
Explain the word without saying it.
```

Не надо делать все раунды “объяснение / одно слово / жесты”. Если успеешь, добавишь tabs.

### UI

```txt
Current team: Team A
Timer: 45
Word: "Nomad"
[Guessed] [Skip]
Score: Team A 5 - Team B 3
AI Host says: Explain fast, no root words!
```

### AI use

AI генерирует слова под культуру:

```txt
Generate 30 party-game words inspired by Kazakhstan, Japan, Brazil, Italy.
```

Fallback: локальный массив слов.

---

## 2. Truth or Dare

Это самая быстрая игра для completeness.

### Core loop

```txt
Игрок выбирается по очереди
Выбирает Truth или Dare
AI генерирует вопрос/задание
Игрок выполняет
Next player
```

### UI

```txt
Current player: Aigerim
[Truth] [Dare]
AI Host card:
"What is a small lie you told recently?"
[Done] [Skip] [Too spicy]
```

### Важная safety штука

Добавь кнопки:

```txt
Family-safe
Soft
Chaotic
```

Не делай adult/nsfw. Для хакатона безопаснее.

### AI use

AI генерирует вопросы по vibe:

```txt
Generate a funny but safe truth question for a group of friends.
```

Fallback: 30 вопросов и 30 действий.

---

## 3. Broken Telephone

Это самый запоминающийся visual moment.

### Core loop

```txt
AI генерирует стартовую фразу
Игрок 1 читает и передает устройство
Игрок 2 пишет, что запомнил
Игрок 3 пишет, что понял
В конце показывается transformation chain
```

### UI

```txt
Original:
"The moon stole grandma's bicycle."

Player 1 heard:
"The moon sold grandma's bicycle."

Player 2 heard:
"Grandma opened a moon bike shop."

Final Reveal:
AI Host: The sentence evolved into folklore.
```

### Почему круто на видео

В конце можно показать:

```txt
Original -> Player 1 -> Player 2 -> Final
```

Это выглядит смешно и понятно даже без звука.

### AI use

AI может:

* генерировать стартовую фразу;
* делать AI player response;
* комментировать финальный результат.

# Voice / озвучка

Не делай платную TTS-интеграцию.

Используй браузерный SpeechSynthesis, если успеешь:

```ts
function speak(text: string) {
  if (typeof window === "undefined") return;
  const utterance = new SpeechSynthesisUtterance(text);
  utterance.rate = 1;
  utterance.pitch = 1;
  window.speechSynthesis.speak(utterance);
}
```

Кнопка:

```txt
🔊 Read rules
```

Это даст “AI ведущий говорит” без сложного API.

Если не успеешь, просто сделай:

```txt
AI Host message bubble
```

# 5. Точная AI-роль в каждой из 8 игр

## 1. Asyk Atu

AI не просто “генерит текст”. Он:

```txt
- расставляет цифровые асыки в разные паттерны;
- объявляет challenge: precision, speed, trick shot;
- после броска комментирует точность;
- рассказывает короткий культурный контекст.
```

Пример AI Host:

```txt
“I placed the асыки in a crescent formation. Knock out the center bone first for a bonus.”
```

## 2. Daruma-san ga koronda

AI:

```txt
- играет Oni;
- выбирает скорость фразы;
- делает fake pauses;
- ловит игроков, которые нажали Move слишком поздно;
- после раунда говорит, кто был reckless.
```

## 3. Adedonha / Stop

AI:

```txt
- выбирает букву;
- генерирует категории под страну;
- проверяет ответы;
- спорные ответы помечает как “needs group vote”;
- добавляет cultural bonus.
```

Пример:

```txt
Letter: M
Categories: food, city, animal, celebration, local word
```

## 4. Ayò Ọlọ́pọ́n

AI:

```txt
- играет соперника;
- объясняет ход простыми словами;
- подсвечивает pits;
- предупреждает о starving move;
- дает после игры “wisdom note”.
```

## 5. Antakshari

AI:

```txt
- задает стартовую букву;
- если игрок не знает песню, AI-player может вступить;
- определяет последнюю букву;
- делает раунд “Bollywood / folk / global songs”;
- ведет scoreboard.
```

## 6. Lotería

AI:

```txt
- выступает cantor;
- не просто называет карту, а дает загадку;
- генерирует themed tabla под выбранную страну;
- объявляет победу;
- делает драматичный reveal.
```

Пример:

```txt
“I carry the night on my wings. Mark El Murciélago if you have it.”
```

## 7. Yutnori

AI:

```txt
- бросает virtual yut sticks;
- объясняет результат do/gae/geol/yut/mo;
- советует маршрут;
- может играть второй командой;
- комментирует lucky throw.
```

## 8. Gol ya Pooch

```txt
- играет роль “master of bluff”;

- генерирует fake body-language clue;
- игроки голосуют;
- AI объясняет, как он их обманул.
```

Пример:

```txt
“My left hand trembled because I wanted you to notice it. The gol was in the right hand.”
```

# 6. Реальный multiplayer: архитектура

Для 5 часов бери **Supabase Realtime**, не самописный WebSocket-сервер. Supabase Realtime дает Broadcast для низколатентных сообщений между пользователями и Presence для синхронизации online-состояния. ([Supabase][20])

## Почему Supabase Realtime

```txt
- не нужен отдельный Node WebSocket server;
- работает с Vercel;
- есть presence;
- можно broadcast actions;
- backend-agent может делать параллельно;
- если сломается, остается local mode.
```

## Multiplayer modes

```txt
Local Pass & Play    -> всегда работает
Realtime Room        -> если Supabase env есть
AI Fill Player       -> если людей мало
```

## Room model

```ts
type Room = {
  code: string;
  hostId: string;
  gameId: string;
  cultureId: string;
  status: "lobby" | "playing" | "finished";
  players: Player[];
  gameState: unknown;
};
```

## Схема Supabase

# 7. Голосовой AI-ведущий

Делать двумя слоями.

## Layer 1: Browser SpeechSynthesis

Это must-have. Бесплатно, быстро, без ключей. Web Speech API включает SpeechSynthesis для text-to-speech, а `speechSynthesis.speak()` ставит utterance в очередь произнесения. ([MDN Web Docs][21])

Файл:

```txt
lib/voice/speak.ts
```

```ts
export function speakHost(text: string, lang = "en-US") {
  if (typeof window === "undefined") return;
  if (!("speechSynthesis" in window)) return;

  window.speechSynthesis.cancel();

  const utterance = new SpeechSynthesisUtterance(text);
  utterance.lang = lang;
  utterance.rate = 0.95;
  utterance.pitch = 1;

  window.speechSynthesis.speak(utterance);
}
```

UI:

```txt
[🔊 Host speaks]
[Auto voice: on/off]
```

## Layer 2: ElevenLabs optional

ElevenLabs сейчас имеет free plan с 10k credits/month, а TTS API/pricing pages описывают text-to-speech и per-character pricing; но для хакатона это optional, потому что API key, latency и audio caching могут съесть время. ([ElevenLabs][22])

Если OpenCode успевает:

```txt

```



# 2. Упрощения без потери вау-эффекта

## 1. Asyk Atu

### Проблема

Полная физика броска и столкновений не нужна. Она сожрет время.

### Упрощение

Сделать **precision aim challenge**:

```txt
На экране круг с асыками.
Игрок выбирает силу и направление через drag/click.
После клика считается accuracy score.
Если точность высокая, асык “выбит”.
```

Технически:

```txt
Нет физики.
Есть target position.
Есть click position.
Distance -> accuracy -> hit/miss.
```

### Это честно?

Да. Суть Asyk Atu сохраняется: **меткость, выбивание асыков, круг, precision**. На видео это даже понятнее, чем физика.

### Вау

Показать:

```txt
AI Referee: “Hit the center asyk for a nomad precision bonus.”
Accuracy: 87%
Asyk knocked out!
```

## 2. Yutnori

### Проблема

Полная доска, маршруты, shortcuts, несколько фишек и capture могут разнести код.

### Упрощение

Сделать **Yutnori Sprint**:

```txt
Одна фишка на команду.
Линейный трек из 12 клеток.
AI бросает yut sticks.
Do = 1, Gae = 2, Geol = 3, Yut = 4, Mo = 5.
Кто первый дошел до конца — победил.
```

### Это честно?

Да, потому что ключевая суть сохраняется: **yut sticks определяют движение по board race**. Просто без сложных маршрутов.

### Вау

```txt
AI Host throws the sticks.
Result: YUT!
Move 4 spaces and throw again.
```

Дополнительный ход на `Yut` и `Mo` можно добавить, если быстро.

## 3. Antakshari

### Проблема

Реальная проверка песен, языков и последней буквы может быть грязной.

### Упрощение

Сделать **Antakshari Chain Lite**:

```txt
AI дает стартовую букву.
Игрок вводит название песни или строку.
Система берет последнюю букву введенного текста.
Следующий игрок должен начать с нее.
Если не знает — нажимает AI Player.
AI Player продолжает цепочку.
```

### Это честно?

Да. Основная механика Antakshari: **цепочка песен по последней букве**.

### Что не делать

Не валидировать, настоящая ли песня.
Добавить кнопку:

```txt
Group accepts
Challenge answer
```

Если `Challenge answer`, AI дает мягкий verdict.

### Вау

```ts
export function selectCulturePack({
  playerCount,
  vibe,
  availableCultures
}: SelectCultureInput): CulturePack {
  const filtered = availableCultures.filter((culture) =>
    culture.supportedVibes.includes(vibe)
  );

  const pool = filtered.length ? filtered : availableCultures;
  const index = Math.floor(Math.random() * pool.length);

  return pool[index];
}
```

## Step 4: собирается Discovery Package

Файл:

```txt
lib/globe/buildDiscovery.ts
```

Функция:

```ts
export async function buildDiscovery(input: BuildDiscoveryInput): Promise<GlobeDiscovery>
```

Она делает 3 вещи:

1. Берет локальные игры культуры из registry.
2. Пытается получить AI intro через `/api/ai`.
3. Если AI упал, использует fallback из `CulturePack`.

```ts
export async function buildDiscovery(input: BuildDiscoveryInput): Promise<GlobeDiscovery> {
  const localRecommendations = getLocalGameRecommendations({
    cultureId: input.culture.id,
    playerCount: input.playerCount,
    vibe: input.vibe,
    availableGames: input.availableGames
  });

  const aiResult = await tryGetAiDiscovery({
    culture: input.culture,
    playerCount: input.playerCount,
    vibe: input.vibe,
    recommendations: localRecommendations
  });

  return {
    culture: input.culture,
    recommendations: aiResult?.recommendations?.length
      ? aiResult.recommendations
      : localRecommendations,
    hostIntro: aiResult?.hostIntro ?? input.culture.fallbackHostIntro,
    culturalFact: aiResult?.culturalFact ?? input.culture.facts[0],
    openingLine: aiResult?.openingLine ?? input.culture.fallbackOpeningLine,
    aiProvider: aiResult?.provider ?? "fallback",
    fallback: !aiResult
  };
}
```

```ts
import type { GameRecommendation } from "@/types/ai";
import type { CulturePack, PartyVibe } from "@/types/culture";
import type { GameDefinition } from "@/types/game";

export function buildFallbackRecommendations(input: {
  culture: CulturePack;
  playerCount: number;
  vibe: PartyVibe;
  games: GameDefinition[];
}): GameRecommendation[] {
  return input.games
    .filter((game) => input.culture.featuredGameIds.includes(game.id))
    .filter((game) => {
      return (
        input.playerCount >= game.playerRange.min &&
        input.playerCount <= game.playerRange.max
      );
    })
    .slice(0, 3)
    .map((game, index) => ({
      gameId: game.id,
      title: game.title,
      country: game.country,
      reason: game.mechanicSummary,
      badge:
        game.implementationStatus === "playable"
          ? "Playable now"
          : game.implementationStatus === "lite"
            ? "Lite mode"
            : "Discovery mode",
      priority: 100 - index
    }));
}
```
